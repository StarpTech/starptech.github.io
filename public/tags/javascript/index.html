Standard formatting in Javascript

2017-06-03 :: Dustin Deus
#javascript 
Formatting in Javascript must not be pain. You don't have to use any linting configurations files. No .eslintrc, .jshintrc, or .jscsrc files to manage. Beside linting it also warns you about programmer errors early e.g unused variables, unhandled errors and more. Give StandardJs a try!
